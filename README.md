### Commands

*目前正在进行第三轮重构，所有功能都暂时无法使用。*

**以下功能排列不分优先级：**

- [x] 帮助文档 `.help`
- [x] 退群指令 `.dismiss`
- [x] 版本信息 `.bot`
- [x] 骰子开关 `.bot on/off`
- [ ] 集成群管 `.group`
- [ ] 授权许可 `.authorize`
- [ ] 发送消息 `.send`
- [ ] 规则速查 `.rules`
- [ ] 标准掷骰 `.r`
- [ ] 跑团记录 `.log`
- [ ] 旁观模式 `.ob`
- [ ] 设置默认骰 `.set`
- [ ] 人物作成 `.coc/dnd`
- [ ] 属性记录 `.st`
- [ ] 角色卡记录 `.pc`
- [ ] COC 检定 `.rc`
- [ ] COC 房规 `.setcoc`
- [ ] COC 理智检定 `.sc`
- [ ] COC 成长检定 `.en`
- [ ] DND 先攻 `.ri` 
- [ ] 先攻列表 `.init`
- [ ] WOD 骰池 `.w(w)`
- [ ] 设置昵称 `.nn(n)`
- [ ] 拓展牌堆 `.draw`
- [ ] 分群牌堆 `.deck`
- [ ] 随机姓名 `.name`
- [ ] 今日人品 `.jrrp`
- [ ] 欢迎词 `.welcome`
- [ ] 第三人称动作 `.me `
